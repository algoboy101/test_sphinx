[0293] 翻转游戏
===============

-  https://leetcode-cn.com/problems/flip-game

题目描述
--------

.. raw:: html

   <p>

你和朋友玩一个叫做「翻转游戏」的游戏，游戏规则：给定一个只有 + 和 - 的字符串。你和朋友轮流将 连续
的两个 "++" 反转成 "--"。
当一方无法进行有效的翻转时便意味着游戏结束，则另一方获胜。

.. raw:: html

   </p>

.. raw:: html

   <p>

请你写出一个函数，来计算出每个有效操作后，字符串所有的可能状态。

.. raw:: html

   </p>

.. raw:: html

   <p>

.. raw:: html

     &quot;++--&quot;

   </pre>
