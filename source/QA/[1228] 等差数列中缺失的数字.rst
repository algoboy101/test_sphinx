--------

.. raw:: html

   <p>

有一个数组，其中的值符合等差数列的数值规律，也就是说：

.. raw:: html

   </p>

::

    <li>在&nbsp;<code>0 &lt;= i &lt; arr.length - 1</code>&nbsp;的前提下，<code>arr[i+1] - arr[i]</code>&nbsp;的值都相等。</li>

   </ul>

   <p>

我们会从该数组中删除一个 既不是第一个
也 不是最后一个的值，得到一个新的数组  arr。

   </p>
