题目描述
--------

.. raw:: html

   <p>

在 R 行 C 列的矩阵上，我们从 (r0, c0) 面朝东面开始

.. raw:: html

   </p>

.. raw:: html

   <p>

这里，网格的西北角位于第一行第一列，网格的东南角位于最后一行最后一列。

.. raw:: html

   </p>

.. raw:: html

   <p>

现在，我们以顺时针按螺旋状行走，访问此网格中的每个位置。

.. raw:: html

   </p>

.. raw:: html

   <p>

.. raw:: html

   </p>

.. raw:: html

   <p>

最终，我们到过网格的所有 R \* C 个空间。

.. raw:: html

   </p>

.. raw:: html

   <p>
