[1109] 航班预订统计
===================

-  https://leetcode-cn.com/problems/corporate-flight-bookings

题目描述
--------

.. raw:: html

   <p>

这里有 n 个航班，它们分别从 1 到 n 进行编号。

.. raw:: html

   </p>

.. raw:: html

   <p>

我们这儿有一份航班预订表，表中第 i 条预订记录 bookings[i] = [i, j,
k] 意味着我们在从 i 到 j 的每个航班上预订了 k 个座位。

.. raw:: html

   </p>

.. raw:: html

   <p>

请你返回一个长度为 n
的数组 answer，按航班编号顺序返回每个航班上预订的座位数。

.. raw:: html

   </p>

.. raw:: html

   <p>

 

.. raw:: html

   </p>

.. raw:: html

   <p>

   </p>
