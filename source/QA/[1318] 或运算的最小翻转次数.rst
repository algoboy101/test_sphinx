[1318] 或运算的最小翻转次数
===========================

-  https://leetcode-cn.com/problems/minimum-flips-to-make-a-or-b-equal-to-c

题目描述
--------

.. raw:: html

   <p>

给你三个正整数 a、b 和 c。

.. raw:: html

   </p>

.. raw:: html

   <p>

你可以对 a 和 b 的二进制表示进行位翻转操作，返回能够使按位或运算   a OR
b == c  成立的最小翻转次数。

.. raw:: html

   </p>

.. raw:: html

   <p>

「位翻转操作」是指将一个数的二进制表示任何单个位上的 1 变成 0 或者 0
变成 1 。

   </p>
