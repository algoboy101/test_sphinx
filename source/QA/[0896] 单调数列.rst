[0896] 单调数列
===============

-  https://leetcode-cn.com/problems/monotonic-array

题目描述
--------

.. raw:: html

   <p>

如果数组是单调递增或单调递减的，那么它是单调的。

.. raw:: html

   </p>

.. raw:: html

   <p>

如果对于所有 i <= j，A[i] <= A[j]，那么数组 A 是单调递增的。
如果对于所有 i <= j，A[i]> = A[j]，那么数组 A 是单调递减的。

.. raw:: html

   </p>

.. raw:: html

   <p>

当给定的数组 A 是单调数组时返回 true，否则返回 false。

.. raw:: html

   </p>
