[0934] 最短的桥
===============

-  https://leetcode-cn.com/problems/shortest-bridge

题目描述
--------

.. raw:: html

   <p>

在给定的二维二进制数组 A 中，存在两座岛。（岛是由四面相连的 1
形成的一个最大组。）

.. raw:: html

   </p>

.. raw:: html

   <p>

现在，我们可以将 0 变为 1，以使两座岛连接起来，变成一座岛。

.. raw:: html

   </p>

.. raw:: html

   <p>

   </p>
