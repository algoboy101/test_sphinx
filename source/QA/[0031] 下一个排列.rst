[0031] 下一个排列
=================

-  https://leetcode-cn.com/problems/next-permutation

题目描述
--------

.. raw:: html

   <p>

实现获取下一个排列的函数，算法需要将给定数字序列重新排列成字典序中下一个更大的排列。

.. raw:: html

   </p>

.. raw:: html

   <p>

如果不存在下一个更大的排列，则将数字重新排列成最小的排列（即升序排列）。

.. raw:: html

   </p>

.. raw:: html

   <p>

必须原地修改，只允许使用额外常数空间。

.. raw:: html

   </p>
