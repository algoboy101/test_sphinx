[0561] 数组拆分 I
=================

-  https://leetcode-cn.com/problems/array-partition-i

题目描述
--------

.. raw:: html

   <p>

给定长度为 2n 的数组, 你的任务是将这些数分成 n 对, 例如 (a1, b1), (a2,
b2), ..., (an, bn) ，使得从1 到 n 的 min(ai, bi) 总和最大。

.. raw:: html

   </p>

.. raw:: html

   <p>

示例 1:

.. raw:: html

   </p>

.. raw:: html

   <pre>
   <strong>输入:</strong> [1,4,3,2]

   <strong>输出:</strong> 4
   <strong>解释:</strong> n 等于 2, 最大总和为 4 = min(1, 2) + min(3, 4).
   </pre>
