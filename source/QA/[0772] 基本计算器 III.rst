=====================

题目描述
--------

.. raw:: html

   <p>

实现一个基本的计算器来计算简单的表达式字符串。

.. raw:: html

   </p>

.. raw:: html

   <p>

表达式字符串可以包含左括号 ( 和右括号 )，加号 + 和减号
-，非负 整数和空格 。

.. raw:: html

   </p>

.. raw:: html

   <p>

表达式字符串只包含非负整数， +, -, \*, / 操作符，左括号 ( ，右括号
)和空格 。整数除法需要向下截断。

.. raw:: html

   </p>

.. raw:: html

   <p>

你可以假定给定的字符串总是有效的。所有的中间结果的范围为 [-2147483648,
2147483647]。

.. raw:: html

   </p>

.. raw:: html

   <p>

 

.. raw:: html

   </p>

   <p>

   </p>

   </pre>
