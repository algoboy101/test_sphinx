[0493] 翻转对
=============

-  https://leetcode-cn.com/problems/reverse-pairs

题目描述
--------

.. raw:: html

   <p>

给定一个数组 nums ，如果 i < j 且 nums[i] > 2\*nums[j] 我们就将 (i,
j) 称作一个重要翻转对。

.. raw:: html

   </p>

.. raw:: html

   <p>

你需要返回给定数组中的重要翻转对的数量。

.. raw:: html

   </p>

.. raw:: html

   <p>

示例 1:

.. raw:: html

   </p>

.. raw:: html

   <pre>
   <strong>输入</strong>: [1,3,2,3,1]
   <strong>输出</strong>: 2
   </pre>

   <p>

   </p>

.. raw:: html
