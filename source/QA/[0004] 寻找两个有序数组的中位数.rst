[0004] 寻找两个有序数组的中位数
===============================

-  https://leetcode-cn.com/problems/median-of-two-sorted-arrays

题目描述
--------

.. raw:: html

   <p>

给定两个大小为 m 和 n 的有序数组 nums1 和 nums2。

.. raw:: html

   </p>

.. raw:: html

   <p>

请你找出这两个有序数组的中位数，并且要求算法的时间复杂度为 O(log(m +
n))。

.. raw:: html

   </p>

.. raw:: html

   <p>

你可以假设 nums1 和 nums2 不会同时为空。

.. raw:: html

   </p>

.. raw:: html

   <p>

示例 1:

.. raw:: html

   </p>

.. raw:: html

   <pre>nums1 = [1, 3]
   nums2 = [2]

   则中位数是 2.0
   </pre>

.. raw:: html

   <p>

示例 2:

.. raw:: html

   </p>

.. raw:: html

   <pre>nums1 = [1, 2]
   nums2 = [3, 4]

   则中位数是 (2 + 3)/2 = 2.5
   </pre>
