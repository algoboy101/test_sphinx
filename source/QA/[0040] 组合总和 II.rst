[0040] 组合总和 II
==================

-  https://leetcode-cn.com/problems/combination-sum-ii

题目描述
--------

.. raw:: html

   <p>

给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

.. raw:: html

   </p>

.. raw:: html

   <p>

candidates 中的每个数字在每个组合中只能使用一次。

.. raw:: html

   </p>

.. raw:: html

   <p>

说明：

.. raw:: html

   </p>

.. raw:: html

   <ul>

::

    <li>所有数字（包括目标数）都是正整数。</li>
    <li>解集不能包含重复的组合。&nbsp;</li>

.. raw:: html

   </ul>

.. raw:: html

   <p>
