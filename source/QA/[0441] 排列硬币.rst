[0441] 排列硬币
===============

-  https://leetcode-cn.com/problems/arranging-coins

题目描述
--------

.. raw:: html

   <p>

你总共有 n 枚硬币，你需要将它们摆成一个阶梯形状，第 k 行就必须正好有 k 枚硬币。

.. raw:: html

   </p>

.. raw:: html

   <p>

给定一个数字 n，找出可形成完整阶梯行的总行数。

.. raw:: html

   </p>

.. raw:: html

   <p>

n 是一个非负整数，并且在32位有符号整型的范围内。

.. raw:: html

   </p>

.. raw:: html

   <p>

示例 1:

.. raw:: html

   </p>

.. raw:: html

   <pre>
   n = 5

   硬币可排列成以下几行:
   &curren;
   &curren; &curren;
   &curren; &curren;

   因为第三行不完整，所以返回2.
   </pre>

   <p>

   因为第四行不完整，所以返回3.
