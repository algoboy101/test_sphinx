[0601] 体育馆的人流量
=====================

-  https://leetcode-cn.com/problems/human-traffic-of-stadium

题目描述
--------

.. raw:: html

   <p>

X 市建了一个新的体育馆，每日人流量信息被记录在这三列信息中：序号
(id)、日期 (visit\_date)、 人流量 (people)。

.. raw:: html

   </p>

.. raw:: html

   <p>

请编写一个查询语句，找出人流量的高峰期。高峰期时，至少连续三行记录中的人流量不少于100。

.. raw:: html

   </p>

.. raw:: html

   <p>

例如，表 stadium：

.. raw:: html

   </p>

.. raw:: html

   <pre>+------+------------+-----------+
   | id   | visit_date | people    |
   +------+------------+-----------+
   | 1    | 2017-01-01 | 10        |
   | 2    | 2017-01-02 | 109       |
   | 3    | 2017-01-03 | 150       |
   | 4    | 2017-01-04 | 99        |
   | 5    | 2017-01-05 | 145       |
   | 6    | 2017-01-06 | 1455      |
   | 7    | 2017-01-07 | 199       |
   | 8    | 2017-01-08 | 188       |
   +------+------------+-----------+
   </pre>

.. raw:: html

   <p>

对于上面的示例数据，输出为：

.. raw:: html

   </p>

.. raw:: html

   <pre>+------+------------+-----------+
   | id   | visit_date | people    |
   +------+------------+-----------+
   | 5    | 2017-01-05 | 145       |
   | 6    | 2017-01-06 | 1455      |
   | 7    | 2017-01-07 | 199       |
   | 8    | 2017-01-08 | 188       |
   +------+------------+-----------+
   </pre>

.. raw:: html

   <p>

 

.. raw:: html

   </p>

.. raw:: html

   <p>

提示： 每天只有一行记录，日期随着 id 的增加而增加。

.. raw:: html

   </p>
