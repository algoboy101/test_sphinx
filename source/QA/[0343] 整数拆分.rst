[0343] 整数拆分
===============

-  https://leetcode-cn.com/problems/integer-break

题目描述
--------

.. raw:: html

给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。
返回你可以获得的最大乘积。

.. raw:: html

   </p>

.. raw:: html

   </p>

   <strong>解释: </strong>2 = 1 + 1, 1 &times; 1 = 1。</pre>
