[0640] 求解方程
===============

-  https://leetcode-cn.com/problems/solve-the-equation

题目描述
--------

.. raw:: html

   <p>

求解一个给定的方程，将x以字符串"x=#value"的形式返回。该方程仅包含'+'，'
- '操作，变量 x 和其对应系数。

.. raw:: html

   </p>

.. raw:: html

   <p>

如果方程没有解，请返回“No solution”。

.. raw:: html

   </p>

.. raw:: html

   <p>

如果方程有无限解，则返回“Infinite solutions”。

.. raw:: html

   </p>

.. raw:: html

   <p>

如果方程中只有一个解，要保证返回值 x 是一个整数。

.. raw:: html

   </p>

.. raw:: html

   <p>
