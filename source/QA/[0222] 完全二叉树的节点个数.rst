[0222] 完全二叉树的节点个数
===========================

-  https://leetcode-cn.com/problems/count-complete-tree-nodes

题目描述
--------

.. raw:: html

   <p>

给出一个完全二叉树，求出该树的节点个数。

.. raw:: html

   </p>

.. raw:: html

   <p>

说明：

.. raw:: html

   </p>

.. raw:: html

   <p>

完全二叉树的定义如下：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第
h 层，则该层包含 1~ 2h 个节点。

.. raw:: html

   </p>

.. raw:: html

   <p>

示例:

.. raw:: html

   </p>

.. raw:: html

   <pre><strong>输入:</strong> 
       1
      / \
     2   3
    / \  /
   4  5 6

   <strong>输出:</strong> 6</pre>
