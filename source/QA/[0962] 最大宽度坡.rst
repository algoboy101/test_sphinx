[0962] 最大宽度坡
=================

-  https://leetcode-cn.com/problems/maximum-width-ramp

题目描述
--------

.. raw:: html

   <p>

给定一个整数数组 A，坡是元组 (i, j)，其中  i < j 且 A[i] <=
A[j]。这样的坡的宽度为 j - i。

.. raw:: html

   </p>

.. raw:: html

   <p>

找出 A 中的坡的最大宽度，如果不存在，返回 0 。

.. raw:: html

   </p>

.. raw:: html

   <p>

 

.. raw:: html

   </p>

.. raw:: html

   <p>

示例 1：

.. raw:: html

   </p>

.. raw:: html

   <pre><strong>输入：</strong>[6,0,8,2,1,5]
   <strong>输出：</strong>4
   <strong>解释：</strong>
   最大宽度的坡为 (i, j) = (1, 5): A[1] = 0 且 A[5] = 5.
   </pre>

.. raw:: html

   <p>

   <strong>解释：</strong>
