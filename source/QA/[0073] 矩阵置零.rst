[0073] 矩阵置零
===============

-  https://leetcode-cn.com/problems/set-matrix-zeroes

题目描述
--------

.. raw:: html

   <p>

给定一个 m x n 的矩阵，如果一个元素为
0，则将其所在行和列的所有元素都设为 0。请使用原地算法。

.. raw:: html

   </p>

   &nbsp; [0,0,0],
