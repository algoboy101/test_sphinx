[1017] 负二进制转换
===================

-  https://leetcode-cn.com/problems/convert-to-base-2

题目描述
--------

.. raw:: html

   <p>

给出数字 N，返回由若干 "0" 和 "1"组成的字符串，该字符串为
N 的负二进制（base -2）表示。

.. raw:: html

   </p>

.. raw:: html

   <p>

除非字符串就是 "0"，否则返回的字符串中不能含有前导零。

.. raw:: html

   </p>

.. raw:: html

   <p>

 

.. raw:: html

   </p>

.. raw:: html

   <p>

.. raw:: html

   </p>

   <strong>解释：</strong>(-2) ^ 2 + (-2) ^ 1 = 2
   </pre>
