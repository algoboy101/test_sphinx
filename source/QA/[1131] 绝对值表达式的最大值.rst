[1131] 绝对值表达式的最大值
===========================

-  https://leetcode-cn.com/problems/maximum-of-absolute-value-expression

题目描述
--------

.. raw:: html

   <p>

给你两个长度相等的整数数组，返回下面表达式的最大值：

.. raw:: html

   </p>

.. raw:: html

   <p>

\|arr1[i] - arr1[j]\| + \|arr2[i] - arr2[j]\| + \|i - j\|

.. raw:: html

   </p>

.. raw:: html

   <p>

其中下标 i，j 满足 0 <= i, j < arr1.length。

.. raw:: html

   </p>

   <p>
