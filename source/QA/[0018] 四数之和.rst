[0018] 四数之和
===============

-  https://leetcode-cn.com/problems/4sum

题目描述
--------

.. raw:: html

   <p>

给定一个包含 n
个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素
a，b，c 和 d ，使得 a + b + c +
d 的值与 target 相等？找出所有满足条件且不重复的四元组。

.. raw:: html

   </p>

.. raw:: html

   <p>

注意：

.. raw:: html

   </p>

   </p>

   <p>

.. raw:: html

   </p>
