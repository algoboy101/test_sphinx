   <pre>+-------------+-------+
   | Column Name | Type  |
   +-------------+-------+
   | player_id   | int   |
   | group_id    | int   |
   +-------------+-------+
   玩家 ID 是此表的主键。
   此表的每一行表示每个玩家的组。
   </pre>

Matches 赛事表

.. raw:: html

   </p>

.. raw:: html

   <pre>+---------------+---------+
   | Column Name   | Type    |
   +---------------+---------+
   | match_id      | int     |
   | first_player  | int     |
   | second_player | int     | 
   | first_score   | int     |
   | second_score  | int     |
   +---------------+---------+
   match_id 是此表的主键。
   每一行是一场比赛的记录，第一名和第二名球员包含每场比赛的球员 ID。
   第一个玩家和第二个玩家的分数分别包含第一个玩家和第二个玩家的分数。
   你可以假设，在每一场比赛中，球员都属于同一组。
   </pre>
