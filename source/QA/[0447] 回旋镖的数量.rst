[0447] 回旋镖的数量
===================

-  https://leetcode-cn.com/problems/number-of-boomerangs

题目描述
--------

.. raw:: html

   <p>

给定平面上 n 对不同的点，“回旋镖” 是由点表示的元组 (i, j,
k) ，其中 i 和 j 之间的距离和 i 和 k 之间的距离相等（需要考虑元组的顺序）。

.. raw:: html

   </p>

.. raw:: html

   <p>

找到所有回旋镖的数量。你可以假设 n 最大为 500，所有点的坐标在闭区间
[-10000, 10000] 中。

.. raw:: html

   </p>

.. raw:: html

   <p>

示例:

.. raw:: html

   </p>

.. raw:: html

   <pre>
   <strong>输入:</strong>
   [[0,0],[1,0],[2,0]]

   <strong>输出:</strong>
   2

   <strong>解释:</strong>
   两个回旋镖为 <strong>[[1,0],[0,0],[2,0]]</strong> 和 <strong>[[1,0],[2,0],[0,0]]</strong>
   </pre>
