题目描述
--------

.. raw:: html

   <p>

给定一个二叉树，返回其节点值自底向上的层次遍历。
（即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历）

.. raw:: html

   </p>

.. raw:: html

   <p>

      / \
     9  20
       /  \
      15   7
   </pre>

.. raw:: html

   <p>

返回其自底向上的层次遍历为：

.. raw:: html

   </p>

.. raw:: html

   <pre>[
     [15,7],
