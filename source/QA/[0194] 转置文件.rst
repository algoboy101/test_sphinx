[0194] 转置文件
===============

-  https://leetcode-cn.com/problems/transpose-file

题目描述
--------

.. raw:: html

   <p>

.. raw:: html

   </p>

.. raw:: html

   <p>

.. raw:: html

   </p>

.. raw:: html

   <p>

.. raw:: html

.. raw:: html

   <p>

   </p>

   <pre>name alice ryan
   age 21 30
   </pre>
