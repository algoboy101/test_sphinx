[0483] 最小好进制
=================

-  https://leetcode-cn.com/problems/smallest-good-base

题目描述
--------

.. raw:: html

   <p>

对于给定的整数 n,
如果n的k（k>=2）进制数的所有数位全为1，则称 k（k>=2）是 n 的一个好进制。

.. raw:: html

   </p>

.. raw:: html

   <p>

   </p>
