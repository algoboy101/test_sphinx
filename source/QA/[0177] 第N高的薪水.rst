[0177] 第N高的薪水
==================

-  https://leetcode-cn.com/problems/nth-highest-salary

题目描述
--------

.. raw:: html

   <p>

编写一个 SQL 查询，获取 Employee 表中第 n 高的薪水（Salary）。

.. raw:: html

   </p>

.. raw:: html

   <pre>+----+--------+
   | Id | Salary |
   +----+--------+
   | 1  | 100    |
   | 2  | 200    |
   | 3  | 300    |
   +----+--------+
   </pre>

.. raw:: html

   <p>

例如上述 Employee 表，n =
2 时，应返回第二高的薪水 200。如果不存在第 n 高的薪水，那么查询应返回 null。

.. raw:: html

   </p>

.. raw:: html

   <pre>+------------------------+
   | getNthHighestSalary(2) |
   +------------------------+
   | 200                    |
   +------------------------+
   </pre>
