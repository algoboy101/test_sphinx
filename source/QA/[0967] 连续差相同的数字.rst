[0967] 连续差相同的数字
=======================

-  https://leetcode-cn.com/problems/numbers-with-same-consecutive-differences

题目描述
--------

.. raw:: html

   <p>

返回所有长度为 N 且满足其每两个连续位上的数字之间的差的绝对值为
K 的非负整数。

.. raw:: html

   </p>

.. raw:: html

   <p>

请注意，除了数字 0
本身之外，答案中的每个数字都不能有前导零。例如，01 因为有一个前导零，所以是无效的；但
0 是有效的。

.. raw:: html

   </p>

.. raw:: html

   <p>

你可以按任何顺序返回答案。
