[0360] 有序转化数组
===================

-  https://leetcode-cn.com/problems/sort-transformed-array

题目描述
--------

.. raw:: html

   <p>

给你一个已经 排好序 的整数数组 nums 和整数 a、b、c。对于数组中的每一个数
x，计算函数值 f(x) = ax2 + bx + c，请将函数值产生的数组返回。

.. raw:: html

   </p>

.. raw:: html

   <p>

要注意，返回的这个数组必须按照
升序排列，并且我们所期望的解法时间复杂度为 O(n)。

.. raw:: html

   </p>

   <p>
