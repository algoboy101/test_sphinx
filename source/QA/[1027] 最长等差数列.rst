[1027] 最长等差数列
===================

-  https://leetcode-cn.com/problems/longest-arithmetic-sequence

题目描述
--------

.. raw:: html

   <p>

给定一个整数数组 A，返回 A 中最长等差子序列的长度。

.. raw:: html

   </p>

.. raw:: html

   <p>

回想一下，A 的子序列是列表 A[i\_1], A[i\_2], ..., A[i\_k] 其中 0 <= i\_1
< i\_2 < ... < i\_k <= A.length - 1。并且如果 B[i+1] - B[i]( 0 <= i <
B.length - 1) 的值都相同，那么序列 B 是等差的。

.. raw:: html

   </p>

   <p>

   </p>

   <p>
