--------

.. raw:: html

   <p>

设计并实现一个 TwoSum 的类，使该类需要支持 add 和 find 的操作。

.. raw:: html

   </p>

.. raw:: html

   <p>

add 操作 -  对内部数据结构增加一个数。 find 操作 -
寻找内部数据结构中是否存在一对整数，使得两数之和与给定的数相等。

   </p>
