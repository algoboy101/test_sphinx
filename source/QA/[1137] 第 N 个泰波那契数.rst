.. raw:: html

泰波那契序列 Tn 定义如下： 

.. raw:: html

   </p>

.. raw:: html

T0 = 0, T1 = 1, T2 = 1, 且在 n >= 0 的条件下 Tn+3 = Tn + Tn+1 + Tn+2

.. raw:: html

   </p>

.. raw:: html

   <p>

给你整数 n，请返回第 n 个泰波那契数 Tn 的值。

   </p>
