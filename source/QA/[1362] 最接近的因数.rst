[1362] 最接近的因数
===================

-  https://leetcode-cn.com/problems/closest-divisors

题目描述
--------

.. raw:: html

   <p>

给你一个整数 num，请你找出同时满足下面全部要求的两个整数：

.. raw:: html

   </p>

.. raw:: html

   <ul>

::

    <li>两数乘积等于 &nbsp;<code>num + 1</code>&nbsp;或&nbsp;<code>num + 2</code></li>
    <li>以绝对差进行度量，两数大小最接近</li>

.. raw:: html

   </ul>

.. raw:: html

   <p>

你可以按任意顺序返回这两个整数。

.. raw:: html

   </p>
