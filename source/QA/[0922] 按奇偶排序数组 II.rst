[0922] 按奇偶排序数组 II
========================

-  https://leetcode-cn.com/problems/sort-array-by-parity-ii

题目描述
--------

.. raw:: html

   <p>

给定一个非负整数数组 A， A 中一半整数是奇数，一半整数是偶数。

.. raw:: html

   </p>

.. raw:: html

   <p>

对数组进行排序，以便当 A[i] 为奇数时，i 也是奇数；当 A[i] 为偶数时， i
也是偶数。

.. raw:: html

   </p>

.. raw:: html

   <p>

你可以返回任何满足上述条件的数组作为答案。

.. raw:: html

   </p>

.. raw:: html

   <p>

 

.. raw:: html

   </p>

.. raw:: html

   <p>
